# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-16

### Added

- Initial release
- 10 built-in icons: ArrowRight, ArrowLeft, ArrowUp, ArrowDown, Check, X, Plus, Minus, Search, Menu
- Base Icon component with customizable size, color, and className props
- CSS animations: spin, pulse, bounce, shake, ping
- Framer Motion integration with AnimatedIcon component
- TypeScript type definitions
- Tree-shaking support
- ESM and CommonJS dual exports
- Example app with Vite
- SVG to React component generator script

### Features

- Full TypeScript support
- Accessible with ARIA labels
- Zero dependencies (except React)
- Optional Framer Motion peer dependency
- CSS animations via separate stylesheet
